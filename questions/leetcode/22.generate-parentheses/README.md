# 22.括号生成

<https://leetcode-cn.com/problems/generate-parentheses/description/>

数字 `n`代表生成括号的对数，请你设计一个函数，用于能够生成所有可能的并且 有效的 括号组合。

示例 1：

```txt
输入：n = 3
输出：["((()))","(()())","(())()","()(())","()()()"]
```

示例 2：

```txt
输入：n = 1
输出：["()"]
```

提示：

- `1 <= n <= 8`
